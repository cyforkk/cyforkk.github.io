## 为什么需要数据统计

### 1. 了解访问情况

通过数据统计，你可以知道：

- 📊 **访问量**：每天有多少人访问
- 👥 **访客数**：独立访客数量
- 📄 **页面浏览量**：总共浏览了多少页面
- ⏱️ **停留时间**：用户在网站停留多久
- 📱 **设备类型**：PC、手机、平板的比例

### 2. 分析用户行为

深入了解用户：

- 🔍 **来源渠道**：搜索引擎、社交媒体、直接访问
- 🌍 **地理位置**：访客来自哪些国家/地区
- 🔗 **热门内容**：哪些文章最受欢迎
- 🚪 **跳出率**：用户是否只看一页就离开
- 🛤️ **访问路径**：用户的浏览轨迹

### 3. 优化网站内容

基于数据做决策：

- ✍️ **内容方向**：根据热门文章调整写作方向
- 🎯 **SEO 优化**：了解搜索关键词，优化 SEO
- 🔧 **用户体验**：发现问题页面，改进体验
- 📈 **增长策略**：分析流量来源，制定推广策略

## 百度统计配置

### 1. 注册百度统计

#### 步骤 1：访问百度统计

打开浏览器，访问：https://tongji.baidu.com/

#### 步骤 2：登录百度账号

- 如果有百度账号，直接登录
- 如果没有，点击「立即注册」创建账号

#### 步骤 3：添加网站

1. 登录后，点击「管理」→「新增网站」
2. 填写网站信息：
   - **网站域名**：你的博客域名（如：cyforkk.top）
   - **网站名称**：你的博客名称
   - **网站首页**：完整的网站地址（如：https://cyforkk.top）
   - **行业类别**：选择「博客」或「个人网站」
3. 点击「确定」

### 2. 获取统计代码

#### 步骤 1：进入代码获取页面

添加网站后，会自动跳转到「代码获取」页面。

如果没有跳转，可以：
1. 点击「管理」→「代码管理」→「代码获取」

#### 步骤 2：复制统计代码

你会看到类似这样的代码：

```html
<script>
var _hmt = _hmt || [];
(function() {
  var hm = document.createElement("script");
  hm.src = "https://hm.baidu.com/hm.js?你的统计ID";
  var s = document.getElementsByTagName("script")[0];
  s.parentNode.insertBefore(hm, s);
})();
</script>
```

**重点**：找到 `hm.js?` 后面的那串字符，这就是你的**百度统计 ID**。

例如：`1234567890abcdef1234567890abcdef`

### 3. 配置 Butterfly 主题

#### 方法 1：使用主题配置（推荐）

编辑 `_config.butterfly.yml` 文件，找到 `baidu_analytics` 配置项：

```yaml
# 百度统计配置
# 官方网站：https://tongji.baidu.com/web/welcome/login
baidu_analytics: 1234567890abcdef1234567890abcdef
```

**注意**：只需要填写统计 ID，不需要完整的代码！

#### 方法 2：手动注入代码

如果方法 1 不生效，可以手动注入：

编辑 `_config.butterfly.yml`，在 `inject` 部分添加：

```yaml
inject:
  head:
    - <script>var _hmt = _hmt || [];(function() {var hm = document.createElement("script");hm.src = "https://hm.baidu.com/hm.js?你的统计ID";var s = document.getElementsByTagName("script")[0];s.parentNode.insertBefore(hm, s);})();</script>
```

### 4. 验证安装

#### 步骤 1：生成并部署网站

```bash
hexo clean
hexo generate
hexo deploy
```

#### 步骤 2：检查代码安装

1. 访问你的博客
2. 按 F12 打开开发者工具
3. 切换到「Network」标签
4. 刷新页面
5. 搜索 `hm.baidu.com`，如果看到请求成功，说明安装正确

#### 步骤 3：等待数据

百度统计通常需要 **20 分钟到 24 小时** 才能看到数据。

回到百度统计后台，点击「报告」→「实时访客」，查看是否有数据。

### 5. 百度统计功能

#### 实时访客

查看当前正在访问的用户：
- 访问时间
- 访问页面
- 来源网站
- 地理位置

#### 趋势分析

查看历史数据趋势：
- 浏览量（PV）
- 访客数（UV）
- IP 数
- 跳出率
- 平均访问时长

#### 来源分析

了解流量来源：
- 搜索引擎（百度、Google、搜狗等）
- 外部链接
- 直接访问
- 搜索词

#### 访问分析

分析用户行为：
- 受访页面
- 入口页面
- 退出页面
- 访问时长
- 访问深度

## Google Analytics 配置

### 1. 注册 Google Analytics

#### 步骤 1：访问 Google Analytics

打开浏览器，访问：https://analytics.google.com/

#### 步骤 2：登录 Google 账号

使用你的 Google 账号登录（Gmail 账号）。

#### 步骤 3：创建账号

1. 点击「开始衡量」
2. 填写账号名称（如：我的博客）
3. 选择数据共享设置（建议全选）
4. 点击「下一步」

#### 步骤 4：创建媒体资源

1. 填写媒体资源名称（如：cyforkk 博客）
2. 选择时区和货币
3. 点击「下一步」

#### 步骤 5：填写业务信息

1. 选择行业类别（如：艺术与娱乐）
2. 选择企业规模（如：小型）
3. 选择使用目的（如：衡量网站流量）
4. 点击「创建」

#### 步骤 6：接受服务条款

勾选「我接受」，点击「接受」。

### 2. 设置数据流

#### 步骤 1：选择平台

选择「网站」。

#### 步骤 2：填写网站信息

1. **网站网址**：https://cyforkk.top
2. **数据流名称**：cyforkk 博客
3. 点击「创建数据流」

#### 步骤 3：获取衡量 ID

创建成功后，你会看到：

```
衡量 ID: G-XXXXXXXXXX
```

这就是你的 **Google Analytics 衡量 ID**。

### 3. 配置 Butterfly 主题

#### 方法 1：使用主题配置（推荐）

编辑 `_config.butterfly.yml` 文件，找到 `google_analytics` 配置项：

```yaml
# 谷歌分析配置
# 官方网站：https://analytics.google.com/analytics/web/
google_analytics: G-XXXXXXXXXX
```

**注意**：填写完整的衡量 ID，包括 `G-` 前缀！

#### 方法 2：使用 Google Tag Manager

如果你使用 Google Tag Manager，可以配置：

```yaml
# https://www.googletagmanager.com/
google_tag_manager:
  tag_id: GTM-XXXXXXX
```

### 4. 验证安装

#### 步骤 1：生成并部署网站

```bash
hexo clean
hexo generate
hexo deploy
```

#### 步骤 2：实时测试

1. 访问你的博客
2. 打开 Google Analytics 后台
3. 点击「报告」→「实时」→「概览」
4. 如果看到有 1 个活跃用户，说明安装成功

#### 步骤 3：检查代码

按 F12 打开开发者工具，在「Network」标签中搜索 `google-analytics.com` 或 `gtag`，确认请求成功。

### 5. Google Analytics 功能

#### 实时报告

查看当前访问情况：
- 活跃用户数
- 每分钟浏览量
- 热门页面
- 流量来源

#### 受众群体报告

了解用户特征：
- 年龄和性别
- 兴趣爱好
- 地理位置
- 使用的设备和浏览器

#### 流量获取报告

分析流量来源：
- 自然搜索（SEO）
- 付费搜索（广告）
- 社交媒体
- 直接访问
- 引荐流量

#### 互动度报告

分析用户行为：
- 浏览量
- 事件数
- 转化次数
- 参与度

## 其他统计工具

### 1. 不蒜子统计

Butterfly 主题内置了不蒜子统计，无需配置即可使用。

**配置位置**：`_config.butterfly.yml`

```yaml
# 不蒜子 PV / UV 统计
busuanzi:
  # 网站 UV 统计
  site_uv: true
  # 网站 PV 统计
  site_pv: true
  # 页面 PV 统计
  page_pv: true
```

**优点**：
- ✅ 免费
- ✅ 无需注册
- ✅ 配置简单
- ✅ 实时显示

**缺点**：
- ❌ 功能简单
- ❌ 无详细分析
- ❌ 数据不可导出

### 2. Microsoft Clarity

微软推出的免费热力图工具。

**官网**：https://clarity.microsoft.com/

**配置**：

```yaml
# Microsoft Clarity 分析配置
microsoft_clarity: 你的项目ID
```

**特色功能**：
- 🎥 会话录制（录制用户操作）
- 🔥 热力图（显示点击热点）
- 📊 滚动深度分析
- 🆓 完全免费

### 3. Cloudflare Analytics

如果你使用 Cloudflare CDN，可以启用其分析功能。

**配置**：

```yaml
# Cloudflare 分析配置
cloudflare_analytics: 你的token
```

**优点**：
- ✅ 隐私友好（不使用 Cookie）
- ✅ 不影响性能
- ✅ 与 CDN 集成

### 4. Umami Analytics

开源的隐私友好统计工具。

**官网**：https://umami.is/

**配置**：

```yaml
umami_analytics:
  enable: true
  serverURL: https://你的umami服务器
  website_id: 你的网站ID
```

**优点**：
- ✅ 开源免费
- ✅ 隐私友好
- ✅ 可自托管
- ✅ 界面简洁

## 数据分析技巧

### 1. 关注核心指标

**必看指标** **PV（浏览量）**：总访问次数
- 👥 **UV（独立访客）**：不重复的访客数
- ⏱️ **平均停留时间**：用户在网站停留多久
- 🚪 **跳出率**：只看一页就离开的比例
- 📄 **平均访问深度**：平均浏览几个页面

### 2. 分析流量来源

**重点关注**：
- 🔍 **搜索引擎流量**：SEO 效果如何
- 🔗 **外部链接**：哪些网站给你带来流量
- 📱 **社交媒体**：社交推广效果
- 🔖 **直接访问**：老用户回访情况

### 3. 优化热门内容

**策略**：
1. 找出访问量最高的文章
2. 分析这些文章的共同特点
3. 创作类似主题的内容
4. 在热门文章中推荐相关文章

### 4. 改进低效页面

**方法**：
1. 找出跳出率高的页面
2. 分析原因（内容质量、加载速度、排版等）
3. 优化这些页面
4. 添加相关文章推荐

### 5. 了解用户需求

**通过搜索词分析**：
1. 查看用户搜索什么关键词来到你的博客
2. 了解用户的需求和痛点
3. 创作针对这些需求的内容
4. 优化现有文章的 SEO

## 常见问题

### 1. 百度统计没有数据？

**可能原因**：
- 刚安装，需要等待 20 分钟到 24 小时
- 统计 ID 填写错误
- 代码未正确安装
- 网站访问量太低

**解决方法**：
1. 检查统计 ID 是否正确
2. 按 F12 检查代码是否加载
3. 自己访问网站测试
4. 等待一段时间

### 2. Google Analytics 显示 0 用户？

**可能原因**：
- 衡量 ID 填写错误
- 广告拦截插件阻止了 GA
- 数据处理延迟

**解决方法**：
1. 检查衡量 ID 格式（G-XXXXXXXXXX）
2. 关闭广告拦截插件测试
3. 使用「实时」报告查看
4. 等待 24-48 小时

### 3. 数据不准确？

**常见情况**：
- 自己的访问被统计
- 爬虫访问被统计
- 多个统计工具数据不一致

**解决方法**：
1. 在百度统计中排除自己的 IP
2. 使用 robots.txt 阻止爬虫
3. 以一个工具的数据为准

### 4. 如何排除自己的访问？

**百度统计**：
1. 进入「管理」→「过滤规则」
2. 添加 IP 排除规则
3. 输入你的 IP 地址

**Google Analytics**：
1. 进入「管理」→「数据流」→「配置标记设置」
2. 添加「排除内部流量」规则
3. 输入你的 IP 地址
